Markdown 富文本渲染

高优先

内联可执行动作

高优先
AI 回复中嵌入可点击的 Action Button。比如提取出待办后，直接附带「添加到待办」按钮；总结完成后附带「保存为日志」按钮。一键执行，无需复制粘贴。

/

斜杠快捷命令

中优先
在输入框键入 / 弹出命令面板：/summarize 总结当前日志、/todos 提取待办、/translate 翻译、/rewrite 改写。减少自然语言描述的成本。

/translate — 翻译为英文
📎

上下文自动注入

高优先
对话框感知当前选中的日志条目。打开对话时自动将当前日志内容作为上下文传入，AI 无需用户重复描述就能理解「这篇日志」指的是什么。还可以支持拖拽文件到对话框追加上下文。
C · 形态层探索 — 对话框本身的进化
◫

侧边栏 vs 弹窗

中优先
当前是居中弹窗（Modal）。另一种形态是右侧滑出面板（Side Panel），可以一边看日志一边对话，不遮挡主内容。适合「边读边问」的场景。

会话持久化与历史

中优先
当前会话关闭即消失。可以将对话历史保存到日志的 sidecar 文件中（如 .conversation.json），下次打开同一篇日志时恢复上下文。或提供全局会话列表。
