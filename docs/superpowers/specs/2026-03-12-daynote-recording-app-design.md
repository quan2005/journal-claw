# DayNote 录音 App — 设计文档

**日期：** 2026-03-12
**状态：** 已确认

---

## 概述

DayNote 是一款极简 macOS 桌面录音应用。核心交互只有一个录音按钮，录音记录按时间自动归档，无任何多余功能。

---

## 技术栈

| 项目 | 选型 |
|------|------|
| 框架 | Tauri v2 |
| 前端 | React + TypeScript |
| 后端 | Rust（Tauri 命令） |
| 渲染 | macOS 系统 WebView（WKWebView） |
| 录音格式 | M4A / AAC |
| 存储位置 | App 私有目录（`~/Library/Application Support/daynote/`） |
| 主题 | 跟随 macOS 系统 Light / Dark 自动切换 |

---

## 状态与交互

### 空闲状态

- 标题栏显示「DayNote」
- 底部显示红色圆形录音键（内含白色实心圆），带微弱呼吸光晕
- 列表显示历史录音，按 `yyyyMM` 分组

### 录制中状态

- 标题栏替换为 `● 00:23`（红点闪烁 + 计时器，tabular-nums）
- 列表顶部自动插入当前录音条目，高亮显示，时长实时更新
- 底部按钮变为红色圆形停止键（内含白色圆角方块）
- 无任何提示文字

### 右键菜单

右键点击列表条目，弹出系统风格上下文菜单：

1. 播放
2. 在 Finder 中显示
3. —（分割线）
4. 删除（红色）

左键点击列表条目：无反应。

## 录音文件

- **文件名格式：** `录音 YYYY-MM-DD HH:mm.m4a`（例：`录音 2026-03-12 22:41.m4a`）
- **存储路径：** `~/Library/Application Support/daynote/`
- **录音格式：** M4A（AAC 编码）
- **元数据来源：** 文件名即为全部元数据，无额外数据库

---

## 列表分组规则

- 按文件名中的年月（`yyyyMM`）分组
- 组内按时间倒序（最新在上）
- 分割线样式：全大写灰色小字，如 `2026 · 03`
- 录制中的条目始终置顶，属于当前月份组

## 主题

跟随 macOS `prefers-color-scheme` 自动切换，无手动开关。

| 元素 | Light | Dark |
|------|-------|------|
| 背景 | `#ffffff` | `#1c1c1e` |
| 标题栏背景 | `#f2f2f7` | `#2c2c2e` |
| 分割线 | `#e5e5ea` | `#3a3a3c` |
| 月份标签 | `#c7c7cc` | `#48484a` |
| 条目文字 | `#1c1c1e` | `#e8e8e8` |
| 时长文字 | `#c7c7cc` | `#48484a` |
| 录音键 | `#ff3b30` | `#ff375f` |
| 录制高亮 | `rgba(255,59,48,0.06)` | `rgba(255,55,95,0.06)` |

## 组件划分

| 组件 | 职责 |
|------|------|
| `App` | 根组件，监听系统主题，持有录音状态 |
| `TitleBar` | 自定义标题栏，空闲显示 App 名，录制中显示计时器 |
| `RecordingList` | 录音列表，按月分组渲染，处理右键菜单 |
| `MonthDivider` | 月份分割线组件 |
| `RecordingItem` | 单条录音行，含高亮状态 |
| `ContextMenu` | 右键上下文菜单（播放 / Finder / 删除） |
| `RecordButton` | 底部录音/停止切换按钮 |
| Rust 命令 | `start_recording`、`stop_recording`、`list_recordings`、`delete_recording`、`reveal_in_finder`、`play_recording` |
