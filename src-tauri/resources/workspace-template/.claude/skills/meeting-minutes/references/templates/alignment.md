# 对齐记录模板（对齐会 / 碰头会）

**适用场景**：2-4 人小范围的需求对齐或方案碰撞。

**核心价值**：记录"对齐了什么、没对齐什么"。不光要记已达成的共识，没达成的分歧更关键 — 它们是下一步推进的发力点。

## 结构

```
## 背景

一句话说明这次对齐的目的和触发原因。

## 参会人

列出参会人及其角色 / 立场。

## 已对齐事项

按议题分节，每个议题：
- **结论**：一句话
- **关键依据**：为什么这么决定（如有争论，简要记录）
- **负责人**：谁来执行

## 未对齐事项

明确列出哪些问题没有达成共识：
- **分歧点**：各方的不同看法
- **下一步**：如何推进（再开会？找谁拍板？先试点验证？）

## 待办
```

## 对齐记录的关键原则

- **"未对齐"比"已对齐"更重要** — 已对齐的事项执行就好，未对齐的才需要持续关注
- **记录每个结论的依据，哪怕只有一句话** — 避免后续有人说"当时没说过这个"
